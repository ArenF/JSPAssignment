# JSPAssignment
JSP 과제용 프로젝트

> 프로젝트 기획은 PLAN.md 파일의 내용을 참고해주세요.

## 목적
- TODO 리스트를 제작해 유저별로 할 일을 관리하고 다루려 함.

## 기능
- 메인 페이지
  - 전체 페이지 이동 기능
- 로그인 페이지
  - 로그인 및 회원가입 기능
- 할 일(TODO) 페이지
  - 할 일 제작 페이지 이동 기능
  - 할 일 목록 조회 기능
  - 할 일 목록 관리 기능(추가, 삭제 및 상태 수정)
